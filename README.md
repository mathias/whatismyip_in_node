What is it?
===========

"What is my IP?"-type service

Written in node.js

Fun little hack :-)

Usage:
======

    $ node app.js

Exercise of getting it up on your webserver is left to the reader.

Then you can simply:

    $ curl whatismyip.example.com:1337

Enjoy!
